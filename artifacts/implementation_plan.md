# 寺院ウェブサイトリニューアル計画

現在の `temple-page` を基に、最新のデザインを取り入れた `temple-page-ai` プロジェクトを作成します。

## ユーザーレビュー確認事項

> [!IMPORTANT]
> **デザインの方向性について**
> 「最新のデザイン」として、以下の要素を取り入れたプレミアムな和モダンデザインを提案します。
> - **洗練されたタイポグラフィ**: `Zen Old Mincho` (Google Fonts) を継続利用しつつ、可読性と美しさを向上させます。
> - **マイクロインタラクション**: ボタンのホバー効果や、スクロールに応じたフェードインアニメーションを追加します。
> - **レスポンシブ対応の強化**: モバイル端末での体験を向上させます。
> - **技術スタック**: Next.js (App Router) + TypeScript + Styled Components (既存との親和性のため) を採用しますが、より最新の構成でセットアップします。

> [!WARNING]
> **お問い合わせフォームについて**
> 外部サービス (AWS API Gateway + SES) との連携部分は、既存のコードロジックを厳密に移植し、挙動を変えないようにします。

## 変更案

### 新規プロジェクトセットアップ

#### [NEW] `temple-page-ai`
- `create-next-app` を使用して新規プロジェクトを作成します。
- `output: "export"` 設定を行い、静的サイトとしての出力を可能にします。
- `styled-components` のセットアップを行います。

### リソース移行

#### [NEW] `public/images`
- `temple-page/src/images` から画像をコピーし、最適化して配置します。
  - `logo.png`
  - `main.webp`, `bellTower.webp`, `mainHall.webp`
  - その他アイコン類

### コンポーネント実装

#### [NEW] デザインシステム
- 色彩パレット（和ごころを感じる色使い）、スペーシング、フォント設定を定義します。

#### [NEW] `src/components`
- **Header/Footer**: ナビゲーションをモダンに刷新します。ハンバーガーメニューのアニメーションを滑らかにします。
- **Hero**: トップページのメインビジュアルを、全画面表示やパララックス効果などで印象的にします。
- **ContactForm**: `src/components/templates/Contact.tsx` のロジック（API呼び出し、バリデーション）を移植しつつ、UIをモダンにスタイリングし直します。

### ページ実装

#### [NEW] `src/app`
- `page.tsx` (Top): ヒーローセクション、お知らせ、寺院紹介へのリンクを配置。
- `about/page.tsx`: 当山について。
- `information/page.tsx`: お知らせ。
- `contact/page.tsx`: お問い合わせ。
- `layout.tsx`: `Zen Old Mincho` フォントの適用、Google Analyticsの設定（既存コードより移植）。

## 検証計画

### 動作確認
- `npm run dev` でローカルサーバーを起動し、各ページの表示崩れがないか確認します。
- レスポンシブ表示（PC、スマホ）を確認します。

### お問い合わせフォーム検証
- フォームの入力バリデーション（必須項目チェック）が正しく動作するか確認します。
- 確認ダイアログ (`window.confirm`) が表示されるか確認します。
- **注意**: 実際に送信ボタンを押すと管理者へメールが飛ぶ可能性があるため、送信直前までの挙動確認にとどめるか、テスト送信であることを明記して実施します。
